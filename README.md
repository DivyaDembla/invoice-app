# Invora — Invoice Management Dashboard

Invora is a modern and responsive invoice management dashboard built using React, Redux Toolkit, Webpack, and Tailwind CSS.
The application allows users to create, manage, track, and monitor invoices through a clean SaaS-inspired user interface.

---

##  Live Demo

https://invoice-app-kappa-gules.vercel.app/

---

# Features

* Responsive SaaS-style dashboard
* Create and manage invoices
* Dynamic revenue overview
* Invoice status tracking
* Mark invoices as paid
* Search and filter invoices
* Invoice details modal
* Real-time form validation
* Local storage persistence
* Mobile responsive design
* Modern and professional UI
